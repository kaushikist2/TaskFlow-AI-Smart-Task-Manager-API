# TaskFlow AI 🚀

A smart task management backend built using **Node.js, Express, and MongoDB**.

## Features

* JWT Authentication
* Task CRUD operations
* Priority & deadline system
* AI-based task suggestions
* RESTful API

## Tech Stack

* Node.js
* Express.js
* MongoDB
* JWT

## API Endpoints

POST /api/auth/register
POST /api/auth/login

GET /api/tasks
POST /api/tasks
PUT /api/tasks/:id
DELETE /api/tasks/:id

## Setup
cd taskflow-ai
npm install
npm run dev
